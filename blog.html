Question-1: What are the different ways to select an element in the DOM?

You can select a element some different ways. You can use method like getElementById() which is only for unique id.You can use getElementsByClassName() for access multiple classes and querySelector("selector") for the first matching element. For multiple matches, querySelectorAll("selector") returns a NodeList.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent are used to access or modify the content of elements in the DOM. innerHTML allows you to retrieve or set the HTML content inside an element, including tags. innerText gives you the text of an element, ignoring hidden elements and styles, while textContent retrieves or sets all the text, including from hidden elements, and is faster than innerText. Generally, innerHTML is used for handling HTML, innerText for visible text.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element instead of individual child elements. This listener then handles events for all of the child elements, even those added dynamically.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in the DOM where an event that is triggered on an element propagates (or "bubbles") up the DOM tree to its parent elements, and then to the root of the document. When an event occurs, the browser first runs the event handler on the target element, and then the event "bubbles" up through each ancestor element, triggering any event listeners attached to them, in the order from the target to the root.
